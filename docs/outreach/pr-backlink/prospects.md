## How to use this table

- **Status** ladder: `idea` → `rules-checked` → `drafted` → `ready-to-send` → (Leif) `submitted` → `won`/`declined`.
- Work top-down by **Priority** (P1 = highest ROI / lowest friction first).
- Before acting on any row, re-confirm the live submission rule (sites change); the
  "Submission rules checked" column is a point-in-time read (June 2026).
- **Never** present a paid listing as an editorial mention. Paid-inclusion rows are
  flagged; only pursue them if Leif explicitly wants paid placement, and disclose.

## Compliance flags (carry into any action)

- **Paid-inclusion / disclose-if-used:** Uneed boost (#3), Tekpon editorial review (#5) — never present as unpaid editorial.
- **Affiliate-driven roundups** (#9, #12, #13): inclusion may hinge on their monetization; never offer payment for an undisclosed placement.
- **People = consent:** rows 15–18 and any HARO follow-up require a lawful basis to
  contact (CAN-SPAM/GDPR) and an opt-out; see pitch templates.
- **No DA "facts":** every DA/DR here is an estimate from SEO-blog research, not measured.
