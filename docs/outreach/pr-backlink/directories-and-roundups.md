- **Skip "submit to 500 directories" paid blasts.** Low-DR spam directories can look

### Roundup outreach rules
- **One genuinely useful, honest paragraph** beats a feature dump. Editors include
  tools their readers will thank them for.
- **Disclose indie status** and the real caveats (no brokerage sync, no predictions).
  Honesty is the differentiator and the credibility.
- **No quid-pro-quo / no undisclosed paid placement.** If a site only adds tools for a
  fee, that's an ad — label it as such if pursued, never as "we got featured."
- **Find the actual editor/contact**; "info@" rarely lands. Use the byline + site contact.
